# merchant-web

Merchant app de quan ly van hanh cho merchant:
- login
- dashboard KPI
- tao shipment
- danh sach shipment + detail + update/cancel
- pickup requests
- tracking noi bo
- change requests
- return requests
- in van don/chung tu
- profile/tai khoan
- notifications

## Run

```bash
npm install
npm run dev
```

Mac dinh chay tai `http://127.0.0.1:5174`.

## Login

Dung tai khoan merchant that da ton tai trong auth-service.

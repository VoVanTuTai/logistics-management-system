Lệnh mới cho Windows:

powershell -ExecutionPolicy Bypass -File .\run-all.windows.ps1 -MobileMode lan
Nếu infra đã chạy rồi, chỉ muốn start lại service/UI:

powershell -ExecutionPolicy Bypass -File .\run-all.windows.ps1 -MobileMode lan -SkipInfra
Nếu chỉ start backend services:

powershell -ExecutionPolicy Bypass -File .\scripts\start-services-retry.windows.ps1
Còn run-all.ps1 bạn có thể giữ cho bản macOS/generic.
